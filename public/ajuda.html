Como funciona

Simples assim: você cola, a gente encurta.

01

Cole sua URL

Na página inicial, tem um campo grande esperando por você. Cola qualquer link lá — pode ser de uma notícia, um vídeo, um arquivo no Google Drive, não importa o tamanho.

O link precisa começar com http:// ou https:// para funcionar.
02

Clique em "encurtar"

Aperta o botão ou pressiona Enter. Em menos de um segundo o link encurtado aparece logo abaixo, pronto pra usar.

Se você encurtar o mesmo link duas vezes, vai receber o mesmo resultado — sem duplicatas.
03

Copie e compartilhe

Clica em "copiar" e o link já vai pro seu Ctrl+V. Compartilha no WhatsApp, email, redes sociais — onde preferir.

Quando alguém abre o link encurtado, é redirecionado automaticamente para a URL original.
04

Encontre seus links depois

Todos os links ficam guardados e você pode acessá-los pela página Meus links. Lá você consegue buscar de três formas diferentes:

  • Pelo código — aqueles 6 caracteres que aparecem no final do link encurtado (ex: aB3xYz)
  • Pela data — escolhe o dia e vê tudo que foi encurtado naquele dia
  • Pelo ID — cada link tem um número único, basta digitá-lo

Usando pela API

Se você quiser usar o encurtador direto pelo terminal ou integrá-lo a outro projeto, aqui vão os comandos:

encurtar uma URL
curl -X POST http://localhost:3000/shorten \
  -H "Content-Type: application/json" \
  -d '{"url":"https://www.exemplo.com"}'
buscar pelo código curto
curl http://localhost:3000/info/aB3xYz
buscar todos os links de uma data
curl "http://localhost:3000/urls?date=2026-02-22"
buscar pelo ID
curl http://localhost:3000/urls/1
